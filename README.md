# DAILY BAKERY BOX
취향 데이터 기반 신선 베이커리 정기구독 서비스

## 프로젝트 한 줄 요약
사용자 취향 데이터를 기반으로 신선한 베이커리를 정해진 날짜에 자동 추천·정기 배송하는 개인화 구독 서비스

## 프로젝트 배경
- 베이커리 소비가 오프라인 중심에서 온라인·구독 중심으로 전환
- ‘빵택배’ 트렌드 확산에도 불구하고 냉동 디저트 위주 시장 구조
- 개인 취향을 반영하지 못하는 구독 서비스

## 문제 정의
- 구독 구성품이 개인 취향과 맞지 않음
- 신선 베이커리 정기 배송 선택지 부족
- 리뷰·피드백이 다음 구독에 반영되지 않음

## 해결 방향
- 취향 설문·행동·리뷰 데이터를 활용한 개인화 추천
- 정해진 배송 스케줄 기반 신선 베이커리 제공
- 리뷰를 다음 추천에 반영하는 피드백 구조 설계

## 서비스 핵심 구조
취향 데이터 수집 → 개인화 추천 → 베이커리 박스 구성 → 정기 배송 → 리뷰 반영

## 핵심 UX Flow
1. 취향 설문
2. 개인화 베이커리 박스 추천
3. 구독 설정 및 정기 배송
4. 리뷰 작성 및 추천 고도화

## 차별화 포인트
- 랜덤이 아닌 취향 기반 자동 구성
- 냉동이 아닌 신선 베이커리 정기 배송
- 사용할수록 정교해지는 추천 구조

## 기대 효과
- 구독 유지율 및 만족도 상승
- 베이커리의 안정적 생산·판매 구조 확보
- 데이터 기반 개인화 서비스 확장 가능성
